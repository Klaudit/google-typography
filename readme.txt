A simple plugin that lets you use and customize (in real-time!) any fonts from Google Fonts on your existing site, all without writing a single line of code.

== Description ==

This plugin allows you to use Google fonts for any text on your existing website. In the Appearance > Typography settings page, simply add and customize any font from the Google font database, then apply the font to any existing HTML tag or CSS selector. Google Typography will do the rest! 

== Installation ==

1. Activate the plugin
2. Navigate to the 'Settings' page
3. Add and customize as many fonts as you'd like
